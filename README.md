# wsl2-rtpro-trouble
Sample project showing troubles with using real time priorities in docker in wsl2

## Dependencies
* Windows 10
* WSL2
* ubuntu 20.04
* docker-compose
* docker (WSL2 integration enabled)

## Build instructions
### host
```bash
make
```

### Docker
```bash
docker-compose build
```

## Run instructions
### host
```bash
./test
```

### Docker
```bash
docker-compose up
```
